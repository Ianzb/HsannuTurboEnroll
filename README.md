## 介绍

东北师大附中自动抢课工具（HsannuTurboEnroll）是用于东北师范大学附属中学校本社团及模块选修课快速抢课的工具，请勿非法使用。

## 运行环境

- Windows 10/11

## 开发环境

- Python 3.12.X
- 运行库详见requirements.txt
